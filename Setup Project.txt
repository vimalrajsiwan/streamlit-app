for this project -- python 3.10.0v
1 STEP 1 Create Python virtual environment:
py -m venv interview_env

2 STEP 2 — Create folder .vscode in your project

Inside your project:

your-project/
   ├── interview_env/
   └── .vscode/
         └── settings.json

         {
  "terminal.integrated.defaultProfile.windows": "Command Prompt",
  "terminal.integrated.profiles.windows": {
    "Command Prompt": {
      "path": "C:\\Windows\\System32\\cmd.exe",
      "args": []
    }
  },
  "python.defaultInterpreterPath": "${workspaceFolder}\\interview_env\\Scripts\\python.exe",
  "python.terminal.activateEnvironment": true
}


Restart VS code -- it will run project in Command Prompt not in PowerSell


Optional
 To Select bdefault 
 Ctrl + Shift + P

 Python => Select Interpreter =>  choose => interview_env


Once activated, you’ll see (interview_env) appear before your command line prompt.
